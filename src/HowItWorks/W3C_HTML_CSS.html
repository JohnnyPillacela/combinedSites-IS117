HTML

Hyper Text Markup Language, also known as HTML, is a document with code that tells a browser how to make a page look like. HTML has many tags built into it which has separate functions. Some tags display text write onto the screen while other tags are used for images or videos. In the case of image and videos, separate requests need to be made to the server to capture this type of media. If there are multiple GET requests from a user for multiple images or videos this can take a long time for a web page to load. A way that websites have been able to speed up the process is by saving cookies on your browser. Cookies are almost like unique identifiers that allow a server to track you better. Any time your browser requests information from a server, it sends cookies along with it, letting the server know that if is you who are requesting something from it and since it has you saved it can send data back even faster.

W3C and CSS

W3C, also known as World Wide Web Consortium, is an organization that maintains the Web and Web standards. It is run by the founder of the World Wide Web, Tim Berners Lee. According to the W3C website, they state that, “W3C's primary activity is to develop protocols and guidelines that ensure long-term growth for the Web. W3C's standards define key parts of what makes the World Wide Web work.” Without the W3C, there wouldn’t be trust in the network as they try to maintain decentralization.
CSS, also known as cascading style sheet, is a programming language that is useful in making a document more presentable. It can let the programmer of the website show others what they want other to see from their page. Unlike HTML which structures a web document, CSS helps style it and add colors, change fonts, and even the layout of the page.
